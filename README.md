# oc-friendly-url
Модуль OpenCart 2 для создания seo url.

**Установка**

Закачать файлы модуля.

**Использование**

1. В меню модули выбрать модуль Friendly-url
2. Выбрать интересующий раздел и нажать сгенерировать url
